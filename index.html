VPS Manager Panel

Single-file PHP server management. Manage VPS, websites, databases, SSL, DNS, and more — all from one lightweight panel.

⭐ View on GitHub 🚀 Quick Install

🚀 One-Command Install

bash <(curl -sSL https://raw.githubusercontent.com/panelboss/vps-manager/main/install.sh)

Runs on http://YOUR_IP:8080 — Ubuntu 20.04 / 22.04 with root access.

✨ 24 Built-in Features

📊 Dashboard

System overview, stats, service status, quick actions at a glance.

🌐 Websites

Create/delete sites with auto Nginx config, DB, and WordPress setup.

🗄️ Databases

CRUD MySQL databases and users, table count, size monitoring.

💾 Backup & Restore

Zip websites + SQL dump, restore from upload — full site snapshots.

⚙️ Services

Install/start/stop Nginx, MySQL, PHP, phpMyAdmin, Certbot in one click.

🐘 PHP Multi-version

PHP 5.6–8.4, switch per website, live progress install, set default CLI.

🔒 SSL

Let's Encrypt via certbot, auto-renew, live progress, per-site status.

📈 Monitor

Real-time CPU, RAM, disk, network, top processes dashboard.

🛡️ Firewall

UFW enable/disable, add/delete rules by port, quick security presets.

📁 File Manager

Browse/edit /var/www, upload files, create folders, delete — all in browser.

📜 Logs

Nginx access/error per site, syslog, PHP-FPM, MySQL logs viewer.

⏰ Cron Jobs

Add/delete cron jobs with common presets for automation.

🚀 Optimize

Auto-tune Nginx + PHP + MySQL + system per RAM tier.

🔀 Redirects

Add/delete 301/302 redirects per domain with auto nginx validation.

🧹 Cache

Purge Nginx fastcgi_cache, enable/disable per site, monitor cache size.

📡 Uptime Monitor

Health check per site, response time, history log, cron every 5 min.

🔐 Security

Fail2Ban install/status, 3 jails, unban IP from panel.

☁️ Cloud Backup

Mega.nz / Google Drive via rclone — auto-sync your backups.

⚡ PHP per Site

Per-website php.ini overrides via .user.ini generation.

🌍 DNS Manager

Cloudflare API: list zones, CRUD A/AAAA/CNAME/MX/TXT/NS records.

🔄 System Migration

Full backup & restore — websites, DBs, Nginx, SSL, cron, UFW. VPS-to-VPS.

💻 Web Terminal

Browser-based bash shell via ttyd — no more PuTTY needed.

👥 Multi-User

User management with admin/user roles, secure authentication.

⚙️ Settings

Change password, system info, panel configuration.
